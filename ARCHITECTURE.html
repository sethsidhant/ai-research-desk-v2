Admin Agents (New) — 4 Phases
🏥
Token Health Check Phase 1
token-health-check.yml · 07:00 AM IST daily
→
Makes live test call to Kite /user/profile
✅ Valid → silent pass
❌ Expired → immediate Telegram alert, 75 min before pipeline fires
Prevents silent pipeline failure on Kite auth errors
💓
Heartbeat Monitor Phase 1
heartbeat-monitor.yml · 09:30 PM IST daily
→
Reads app_settings heartbeat timestamps
Checks: Token refresh · Pipeline · Digest
Flags any job older than expected threshold
Sends formatted ✅/❌ status card to Telegram admin
🔍
Onboarding Watchdog Phase 2
onboarding-watchdog.yml · 08:20 AM IST daily
→
Checks user_stocks + portfolio_holdings
Finds stocks where fundamentals_updated_at = null
Triggers onboardStock.js for each pending stock
Reports count onboarded to Telegram
🔬
Data Quality Agent Phase 3
data-quality-agent.yml · every 2 days · 08:00 PM IST
→
DB Checks: history gaps · score gaps · price staleness · fundamentals age · FII/MF data lag
Screener spot-check: 5 random stocks, compare PE + price vs DB, flag >5% deviation
Sends full quality report card to Telegram
Supabase Database Tables
stocks
Master table. ticker, name, industry, current_price, fundamentals (PE, ROE, ROCE, D/E, promoter), latest_headlines, ai_brief
daily_history
OHLCV per stock per market day. Source: Kite Historical API. Used for RSI, DMA, chart, volume alerts
daily_scores
RSI, PE deviation, above 50/200 DMA, composite score, classification, suggested action — per stock per day
fii_sector
FII fortnight flow by sector (14 sectors). Used for sector correlation on watchlist + portfolio
fii_dii_daily
Daily FII + DII net buy/sell ₹Cr. Used for Market Pulse charts and evening digest
mf_sebi_monthly
Monthly MF equity + debt flows from SEBI. 3-5 day lag. Used for MF Flow chart on Market Pulse
index_history
Nifty 50 + Nifty 500 daily closes. Used for portfolio/watchlist benchmark chart comparison
user_stocks
Watchlist holdings per user. entry_price, invested_amount, added_at. Virtual P&L simulation
portfolio_holdings
Real holdings per user. quantity, avg_price, broker, investment_date, import_source. Actual P&L
user_settings
Per-user BYOK Anthropic API key. Used by /api/stock-brief to use user's own key
user_alert_preferences
RSI thresholds, DMA alerts, digest time, Telegram chat ID, P&L alert toggles
app_settings
kite_access_token, cron heartbeats (pipeline, digest, token refresh last run timestamps)
